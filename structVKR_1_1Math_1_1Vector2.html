Detailed Description

template<typename T>
struct VKR::Math::Vector2< T >

A Two-Component contiguous Vector.

Template Parameters
TThe type to contain within the vector. This is stored as a union, with {x, y}, {u, v} and array index members.
Note
The size of a Vector2 is always sizeof(T) * 2.
Remarks
Data alignment is implementation defined.
