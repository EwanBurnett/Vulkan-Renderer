Detailed Description

template<typename T>
struct VKR::Math::Vector4< T >

A Four-Component continuous Vector.

Template Parameters
TThe type to contain within the vector. This is stored as a union, with {x, y, z, w} and array index members.
Note
The size of a Vector4 is always sizeof(T) * 4.
Remarks
Data alignment is implementation defined.
