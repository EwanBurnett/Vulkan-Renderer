Detailed Description

template<typename T>
struct VKR::Math::Vector3< T >

A Three-Component contiguous Vector.

Template Parameters
TThe type to contain within the vector. This is stored as a union, with {x, y, z}, {u, v, w} and array index members.
Note
The size of a Vector3 is always sizeof(T) * 3.
Remarks
Data alignment is implementation defined.
